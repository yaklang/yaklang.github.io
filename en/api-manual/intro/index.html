Yaklang 内置库参考

欢迎阅读 Yaklang 内置库参考手册。Yaklang 是一门面向 Web 安全研发的垂直领域语言，底层由 Golang 编写并通过统一的导出机制把大量能力封装成"开箱即用"的内置函数与内置库。本手册系统性地整理这些能力，按"先理解、再使用、后查证"的顺序，帮助你从入门到精通地掌握 Yaklang 的标准能力集。

关于本手册

本手册为重构后的新版内容，按主题分章、渐进式编排，便于检索与学习。左侧目录支持搜索，输入关键字即可跨章节（含折叠分类）定位条目。若需查阅历史内容，请展开目录底部的 「（旧：旧版手册）」。完整的逐库逐函数定义请查阅 「内置标准库完全手册」。

Yaklang 的内置能力概览​

Yaklang 的能力由两部分组成：

  • 内置函数（全局函数）：无需引入任何库即可直接调用的全局函数，覆盖类型转换、打印调试、切片与映射操作、并发控制、错误处理等基础编程能力。例如 println、len、append、typeof、desc、die、sleep 等。
  • 内置库（标准库）：以 库名.函数名 形式调用的功能模块，覆盖字符串处理、文件与操作系统编程、网络通信、安全测试、AI 编程等垂直领域。例如 str、file、os、http、poc、fuzz、mitm、ai 等。

内置库大致可以划分为以下能力域：

  • 基础编程：str（字符串）、re/re2（正则）、json/yaml/xml（数据格式）、codec（编解码与加解密）、math、time 等。
  • 操作系统与文件：file、filesys、os、exec、env 等。
  • 网络编程：tcp、udp、tls、dns、subdomain 等。
  • 安全能力：http（标准 HTTP）、fuzz/fuzzx（HTTP 模糊测试）、synscan/servicescan（端口与服务扫描）、crawler/crawlerx（爬虫）、mitm（中间人劫持）、poc（漏洞验证）、nuclei 等（详见「内置标准库完全手册」）。
  • AI 编程：ai（大模型对话、函数调用、结构化流）、rag（知识库检索增强）、aim（ReAct 智能体引擎）、aireducer（大文本切块流水线）等。

本手册结构​

内置函数与内置操作​

讲解无需引库即可使用的全局内置函数，以及类型、切片、映射等内置操作。

Yaklang 字符串操作​

以渐进式方式展示文本处理：从基础的字符串拼接、查找、替换，到分割与格式化，再到高级文本处理技巧。

操作系统与文件编程​

介绍文件读写、目录遍历、路径处理、环境变量与外部命令执行等系统编程能力。

网络编程​

介绍 TCP/UDP 通信、TLS、DNS 解析与子域名收集等底层网络编程能力。

Yaklang 中的 AI 编程​

介绍如何在 Yaklang 中调用大模型：ai 库的对话、函数调用与结构化流，rag 的知识库检索增强，aim 的 ReAct 智能体引擎，以及 aireducer 的大文本切块流水线。

内置标准库完全手册​

由源码注释自动生成的逐库、逐函数完整定义参考，适合精确查证某个 API 的签名、参数与返回值。

阅读建议​

  • 初学者：建议从「内置函数与内置操作」「Yaklang 字符串操作」开始，建立基础编程认知。
  • 系统与网络开发：进入「操作系统与文件编程」「网络编程」，掌握文件、命令执行与 TCP/UDP/TLS/DNS 的写法。
  • AI 应用开发：进入「Yaklang 中的 AI 编程」，掌握大模型对话、结构化抽取、文本切块与智能体编排。
  • 查证 API：在「内置标准库完全手册」中按库名检索具体函数定义。
