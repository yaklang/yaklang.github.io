http

成员函数函数描述/介绍
http.Do执行一个 http 请求，这个请求是 http.NewRequest 创建的
http.Get发送一个 http 请求
http.GetAllBody
http.NewRequest创建一个新的 Request，创建之后，需要通过 http.Do 来发送
http.Post
http.Raw创建一个 Golang 原生的 *http.Request
http.Request立即发起一个 http 请求，不需要 http.Do 来执行
http.body【参数】设置请求的 body
http.cookie【参数】设置请求的 Cookie
http.dump工具函数，把 http.Request/http.Response 变成完整的数据包 []byte
http.dumphead工具函数，把 http.Request/http.Response 的数据包的头序列化程 []byte
http.header【参数】为请求设置 http header
http.json设置发送模式为 JSON，传入的参数会被 JSON 序列化，把结果设置为 Body，然后设置 application/json 为 Content-Type
http.proxy【参数】为请求设置 http 代理
http.redirect设置重定向
http.show调试函数，展示原始数据包内容
http.showhead调试函数，展示原始数据包内容，不包含 body
http.timeout【参数】设置超时时间
http.ua【参数】设置 UserAgent
http.uarand【参数】设置随机 UserAgent
http.useragent【参数】设置 UserAgent

执行一个 http 请求，这个请求是 http.NewRequest 创建的

详细描述#

定义：#

func http.Do(request: *yaklib.yakHttpRequest) return (r0: *http.Response, r1: error)

参数#

参数名参数类型参数解释
request*yaklib.yakHttpRequest

返回值#

返回值(顺序)返回值类型返回值解释
r0*http.Response
r1error

http.redirect#

设置重定向

详细描述#

定义：#

func http.redirect(v1: func (v1: *http.Request, v2: []*http.Request) return(bool) ) return (r0: func httpOption(v1: *yaklib.yakHttpRequest) )

参数#

参数名参数类型参数解释
v1func (v1: *http.Request, v2: []*http.Request) return(bool) 回调函数，第一个参数为当前请求，第二个函数为已经重定向的请求，返回值为是否继续重定向？

返回值#

返回值(顺序)返回值类型返回值解释
r0func httpOption(v1: *yaklib.yakHttpRequest)
