Example:

req, err = http.Raw("GET / HTTP/1.1\r\nHost: www.yaklang.com\r\n\r\n")rsp, err = http.Do(req)

rsp, err = http.Get("http://www.yaklang.com", http.timeout(10))

NewRequest#

详细描述#
